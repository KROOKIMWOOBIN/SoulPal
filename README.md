# ✨ SoulPal — 나만의 AI 친구

> 완전 오프라인 AI 동반자 앱. 인터넷 없이도 작동하며 모든 대화가 내 기기에만 저장됩니다.

---

## 📥 설치 (Android)

### 최신 버전 다운로드

👉 **[Releases 페이지에서 APK 다운로드](https://github.com/KROOKIMWOOBIN/SoulPal/releases/latest)**

| APK 파일 | 기기 |
|----------|------|
| `SoulPal-arm64.apk` | 최신 안드로이드 폰 **← 대부분 이걸로** |
| `SoulPal-arm32.apk` | 구형 안드로이드 폰 |
| `SoulPal-x86_64.apk` | 에뮬레이터 |

### 설치 방법

1. 위 링크에서 APK 다운로드
2. 안드로이드 설정 → **보안** → **출처를 알 수 없는 앱 설치 허용**
3. 다운로드한 파일 탭 → 설치

> **첫 실행 시 AI 모델 약 800MB를 다운로드합니다.**
> Wi-Fi 환경에서 실행을 권장해요.

---

## ✨ 주요 기능

- 🤖 **완전 오프라인 AI** — 인터넷 없이도 대화 가능
- 🔒 **프라이버시 보호** — 모든 대화가 기기에만 저장
- 🎨 **캐릭터 커스터마이징** — 관계, 성격, 말투, 관심사 등 6가지 설정
- 🌙 **다크 모드** 지원
- 🔍 **대화 검색** — 이전 메시지 검색
- ⭐ **즐겨찾기** — 자주 대화하는 친구 고정
- 💬 **AI 응답 재생성** — 마음에 안 들면 다시 생성

## 🛠️ 개발자용 빌드

```bash
git clone https://github.com/KROOKIMWOOBIN/SoulPal.git
cd SoulPal
bash setup.sh
flutter run
```
